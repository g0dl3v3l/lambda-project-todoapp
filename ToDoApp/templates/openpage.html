Features

Everything you need in a simple task scheduler and more.

Schedule

This application has a simple user interface where the user can enter the task discription and also the date at which the task to be completed.

Sort

This application has a sort filter which can filter all the tasks corresponding to the user selected date.

Update

All the scheduled tasks can be updated as the user wishes and any task can be removed by simply clicking delete button.
